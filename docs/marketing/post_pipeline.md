# Post Pipeline

## Purpose

Define the posts that should be created and describe how each post moves from idea to publication.

## Source Of Truth

This file is the source of truth for posts to be created. When planning blog, social, newsletter, or magnet content, use this pipeline to decide what post exists, why it should be made, what state it is in, and what needs to happen next.

If another marketing file suggests a post idea that is not represented here, add it to this pipeline before drafting or publishing it.

## Notes

Track post ideas, approved posts, drafts, reviews, publication status, and follow-up actions here.
