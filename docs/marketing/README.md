# Marketing

## Purpose

This section describes how business truth is turned into public communication.

## What Belongs Here

Keep marketing strategy, brand voice, content workflow, blog, social, newsletter, and magnet guidance here.

## Working Areas

- `blog/` contains blog-specific guidance.
- `blog/posts/` is reserved for draft or published blog post files. It currently contains only a placeholder so the folder remains available in the repository.

## File Order

Read these files in order when building marketing context:

1. `01-strategy.md`
2. `02-brand-voice.md`
3. `post_pipeline.md`
4. `04-blog.md`
5. `05-social.md`
6. `06-newsletter.md`
7. `07-magnets.md`

## How To Use This Section

Use this section before creating, revising, or evaluating marketing material. Start with `01-strategy.md`, then use the channel-specific file for the work being done.
